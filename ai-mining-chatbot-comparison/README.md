# Исследование возможностей чат-ботов с ИИ в добывающей промышленности

## Автор: [Максим]

## Цель исследования
Сравнить качество решения прикладных задач горнорудной отрасли с помощью зарубежного (ChatGPT) и отечественного (GigaChat) ИИ-ассистентов.

## Используемые модели
1. **Зарубежный бот:** ChatGPT-4o
2. **Отечественный бот:** GigaChat-Pro

## Выбранные кейсы
1. **Кейс 1: Диагностика неисправностей насосного оборудования**
2. **Кейс 2: Анализ требований охраны труда**

## Методология
1. Сформулирован единый запрос для каждого кейса
2. Запросы отправлены обоим ботам
3. Результаты сохранены
4. Проведен сравнительный анализ
